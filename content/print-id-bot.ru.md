@print_id_bot — это Telegram-бот, который показывает ваш **Telegram user ID**, **chat_id** (для групп и супергрупп) и ID канала. Работает в личке и в группах, отвечает на 16 языках в зависимости от настроек языка в Telegram.

## Команды

| Команда | Действие |
|---------|----------|
| `/start` | Приветствие, ваш User ID и Chat ID, inline-кнопки /id и /help |
| `/id` | Ваш User ID и Chat ID (краткий ответ) |
| `/help` | Полная справка и список команд |
| `/json` | Текущее сообщение в формате JSON (для разработчиков) |
| `/dump` | Полный разбор обновления и сырой JSON (только в личке) |

В личке отправка любого сообщения (текст, фото, стикер и т.д.) также возвращает ваш User ID и Chat ID. Для пересланных сообщений бот показывает ID автора, если доступен, или «ID hidden», если настройки приватности отправителя это блокируют. Подробнее: [пересланные сообщения и скрытые ID]({{< relref "telegram-forwarded-message-id.md" "en" >}}).

## Поддерживаемые языки (16)

Бот выбирает язык ответа по вашему `language_code` в Telegram. Поддерживаются: английский, русский, персидский, узбекский, хинди, китайский, турецкий, португальский, испанский, арабский, индонезийский, немецкий, урду, французский, тагальский, амхарский. Неподдерживаемые коды переключаются на английский. См. [как работает выбор языка]({{< relref "telegram-id-bot-languages.md" "en" >}}).

## Поведение в группах и супергруппах

В группах бот отвечает только когда вы:

- Отвечаете на сообщение бота
- Отправляете `/id` или `/help` (с @print_id_bot или без)
- Отправляете любую команду с @print_id_bot (например, `/foo@print_id_bot`)
- Упоминаете @print_id_bot в сообщении

В остальных случаях бот молчит. При добавлении в группу он отправляет приветствие с названием группы и chat_id. Краткая инструкция: [добавить бота в группу для получения chat_id]({{< relref "add-bot-to-group-chat-id.md" "en" >}}).

## Функции для разработчиков

- **/json** — Возвращает текущее сообщение в формате JSON.
- **/dump** — Полный разбор обновления и сырой JSON (только в личке). Полезно для отладки Bot API. См. [разбор JSON обновления Telegram]({{< relref "telegram-json-update.md" "en" >}}).

## FAQ

### Работает ли бот в каналах?
Бот работает в личке и в группах/супергруппах. Для каналов нужен ID канала из другого потока (например, публикация от имени канала и бот, получающий обновления). См. [получить ID канала для Bot API]({{< relref "how-to-get-channel-id-bot-api.md" "en" >}}).

### Можно ли получить чужой Telegram user ID?
Бот показывает ваш ID или chat_id чатов, в которых вы участвуете. Для пересланных сообщений — ID отправителя только если он разрешил пересылку; иначе «ID hidden». Нельзя узнать ID по номеру телефона или username. См. [Telegram ID и username]({{< relref "telegram-id-vs-username.md" "en" >}}).

### Почему бот не отвечает в моей группе?
Бот отвечает только при ответе на него, отправке /id или /help, либо упоминании @print_id_bot. См. [бот не отвечает в группе]({{< relref "bot-not-responding-in-group.md" "en" >}}) — полный чеклист.

### Бот бесплатный?
Да. Регистрация и оплата не требуются.

### В чём разница между user ID и chat_id?
**User ID** идентифицирует аккаунт Telegram. **chat_id** — чат (личка, группа, супергруппа, канал). См. [Telegram ID и username]({{< relref "telegram-id-vs-username.md" "en" >}}).
