Deine **Telegram user ID** ist ein numerischer Identifikator für dein Konto. Sie ist stabil, wird einmal vergeben und nirgendwo in der Telegram-App angezeigt. Bots und Integrationen können sie nur lesen, wenn du mit ihnen interagierst — zum Beispiel wenn du einen Bot öffnest und auf Start tippst oder eine Nachricht sendest. Diese Hub-Sammlung enthält Anleitungen, die dir helfen, deine user ID sicher zu finden, zu kopieren und zu verstehen.

Du kannst die ID einer anderen Person nicht allein über Telefonnummer oder Benutzername erhalten. Dienste, die «ID-Suche per Telefon» versprechen, sind oft Betrug oder verletzen die Privatsphäre. IDs werden nur durch Interaktion erhalten: wenn ein Nutzer einen Bot öffnet, eine Nachricht weiterleitet (falls erlaubt) oder an einem Chat teilnimmt, auf den du Zugriff hast.

## FAQ

### Ist meine Telegram ID privat?

Deine Telegram ID wird nicht öffentlich in der Telegram-Oberfläche angezeigt, aber Bots und Apps können sie lesen, wenn du mit ihnen interagierst. Du kontrollierst, wer sie erhält, indem du wählst, welche Bots und Apps du nutzt.

### Kann ich die ID einer anderen Person per Telefonnummer oder Benutzername erhalten?

Nein. Du kannst eine Telegram user ID nicht zuverlässig allein aus Telefonnummer oder Benutzername erhalten. IDs werden nur erhalten, wenn der Nutzer mit einem Bot oder einer App interagiert oder eine Nachricht weiterleitet (und Weiterleitung erlaubt). Meide Dienste, die anderes behaupten.

### Was ist der Unterschied zwischen user ID und chat_id?

**User ID** identifiziert ein Telegram-Konto. **chat_id** identifiziert eine Unterhaltung (privater Chat, Gruppe, Supergruppe, Kanal). In einem privaten Chat sind user ID und chat_id dieselbe Zahl.
