## सबसे तेज़ तरीका (1 सेकंड)

बॉट खोलें और **शुरू करें** दबाएं। यह वापस देगा:

- आपका **Telegram user ID**
- आपका **username** (अगर है तो)
- (ग्रुप में) ग्रुप या सुपरग्रुप का **chat_id**

## विधि 1 — Telegram बॉट का उपयोग करें (अनुशंसित)

1. बॉट खोलें: https://t.me/print_id_bot
2. **शुरू करें** पर टैप करें
3. संदेश से अपना Telegram ID कॉपी करें

**यह विधि सबसे अच्छी क्यों:** सरल है, मोबाइल और डेस्कटॉप पर काम करती है, और डेवलपर टूल की ज़रूरत नहीं।

## विधि 2 — ग्रुप chat_id प्राप्त करें (Bot API के लिए)

अगर आपको इंटीग्रेशन के लिए ग्रुप **chat_id** चाहिए:

1. बॉट को अपने ग्रुप में जोड़ें
2. ग्रुप में कोई भी संदेश भेजें (या बॉट के कमांड जैसे `/id` का उपयोग करें)
3. बॉट ग्रुप का **chat_id** के साथ जवाब देगा

> नोट: ग्रुप और सुपरग्रुप ID अक्सर ऋणात्मक संख्याएँ होती हैं। यह सामान्य है।

## आम समस्याएँ

- **बॉट मेरे ग्रुप में जवाब नहीं देता:** सुनिश्चित करें कि बॉट को संदेश पढ़ने की अनुमति है, और प्राइवेसी मोड सेटिंग्स जांचें (बॉट कैसे बनाया गया है उस पर निर्भर)।
- **मुझे सिर्फ username दिखता है, ID नहीं:** username ID नहीं हैं। ID संख्यात्मक होते हैं।
- **मुझे किसी और का ID चाहिए:** यह टूल अपना ID या उन चैट का chat_id प्राप्त करने के लिए है जिनमें आप शामिल हैं।

## FAQ

### क्या मेरा Telegram ID निजी है?
आपका Telegram ID Telegram UI में सार्वजनिक रूप से नहीं दिखाया जाता, लेकिन बॉट और ऐप इसे तब पढ़ सकते हैं जब आप उनके साथ इंटरैक्ट करते हैं।

### क्या मैं फोन नंबर या username से ID प्राप्त कर सकता हूँ?
आधिकारिक तौर पर, आम तौर पर बिना इंटरैक्शन के सिर्फ फोन नंबर या username से किसी का संख्यात्मक ID विश्वसनीय रूप से प्राप्त नहीं कर सकते। उन सेवाओं से बचें जो «फोन से ID खोज» का वादा करती हैं — वे अक्सर घोटाले या प्राइवेसी उल्लंघन होती हैं।

### user ID और chat_id में क्या अंतर है?
- **User ID** एक Telegram अकाउंट की पहचान करता है।
- **chat_id** एक बातचीत की पहचान करता है (निजी चैट, ग्रुप, सुपरग्रुप, चैनल)।
