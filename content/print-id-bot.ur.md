@print_id_bot ایک Telegram بٹ ہے جو آپ کا **Telegram user ID**، **chat_id** (گروپس اور سپرگروپس کے لیے) اور چینل ID دکھاتا ہے۔ پرائیویٹ چیٹ اور گروپس میں کام کرتا ہے، آپ کی Telegram زبان کی سیٹنگ کے مطابق 16 زبانوں میں جواب دیتا ہے۔

## کمانڈز

| کمانڈ | کام |
|-------|-----|
| `/start` | خوش آمدید پیغام، User ID اور Chat ID، /id اور /help کے لیے ان لائن بٹن |
| `/id` | User ID اور Chat ID (مختصر جواب) |
| `/help` | مکمل تفصیل اور کمانڈز کی فہرست |
| `/json` | موجودہ پیغام JSON میں (ڈویلپرز کے لیے) |
| `/dump` | مکمل اپڈیٹ کی تفصیل اور خام JSON (صرف پرائیویٹ چیٹ) |

پرائیویٹ چیٹ میں کوئی بھی پیغام (ٹیکسٹ، فوٹو، ٹکر وغیرہ) بھیجنے پر User ID اور Chat ID ملتا ہے۔ فارورڈ کیے گئے پیغامات کے لیے بٹ مصنف کا ID دکھاتا ہے جب دستیاب ہو، یا "ID hidden" جب بھیجنے والے کی پرائیویسی سیٹنگز بلاک کرتی ہیں۔ تفصیل: [فارورڈ پیغامات اور چھپے ہوئے ID]({{< relref "telegram-forwarded-message-id.md" "en" >}}).

## معاون زبانیں (16)

بٹ آپ کے Telegram `language_code` سے جواب کی زبان منتخب کرتا ہے۔ معاون: انگریزی، روسی، فارسی، ازبک، ہندی، چینی، ترکی، پرتگالی، ہسپانوی، عربی، انڈونیشیائی، جرمن، اردو، فرانسیسی، ٹیگالوگ، امہاری۔ غیر معاون کوڈز انگریزی پر واپس جاتے ہیں۔ دیکھیں [زبان کا انتخاب کیسے کام کرتا ہے]({{< relref "telegram-id-bot-languages.md" "en" >}}).

## گروپ اور سپرگروپ رویہ

گروپ میں بٹ صرف تب جواب دیتا ہے جب آپ:

- بٹ کے پیغام کا جواب دیتے ہیں
- `/id` یا `/help` بھیجتے ہیں (@print_id_bot کے ساتھ یا بغیر)
- @print_id_bot کے ساتھ کوئی کمانڈ بھیجتے ہیں (مثلاً `/foo@print_id_bot`)
- اپنے پیغام میں @print_id_bot کا ذکر کرتے ہیں

ورنہ خاموش رہتا ہے۔ گروپ میں شامل ہونے پر گروپ نام اور chat_id کے ساتھ خوش آمدید پیغام بھیجتا ہے۔ مختصر گائیڈ: [chat_id کے لیے گروپ میں بٹ شامل کریں]({{< relref "add-bot-to-group-chat-id.md" "en" >}}).

## FAQ

### کیا بٹ چینلز میں کام کرتا ہے؟
بٹ پرائیویٹ چیٹ اور گروپس/سپرگروپس میں کام کرتا ہے۔ چینلز کے لیے چینل ID دوسرے فلو سے چاہیے (مثلاً چینل کے طور پر پوسٹ کرنا اور اپڈیٹس وصول کرنے والا بٹ)۔ دیکھیں [Bot API کے لیے Telegram چینل ID حاصل کریں]({{< relref "how-to-get-channel-id-bot-api.md" "en" >}}).

### کیا میں کسی اور کا Telegram user ID حاصل کر سکتا ہوں؟
بٹ آپ کا اپنا ID یا جن چیٹس میں آپ ہیں ان کا chat_id دکھاتا ہے۔ فارورڈ پیغامات کے لیے بھیجنے والے کا ID صرف تب دکھاتا ہے جب وہ فارورڈ کی اجازت دیں؛ ورنہ "ID hidden"۔ فون نمبر یا username سے ID تلاش نہیں کر سکتے۔ دیکھیں [Telegram ID بمقابلہ username]({{< relref "telegram-id-vs-username.md" "en" >}}).

### بٹ میرے گروپ میں جواب کیوں نہیں دیتا؟
بٹ صرف تب جواب دیتا ہے جب آپ اسے جواب دیتے ہیں، /id یا /help بھیجتے ہیں، یا @print_id_bot کا ذکر کرتے ہیں۔ مکمل چیک لسٹ: [گروپ میں جواب نہ دینے والا بٹ]({{< relref "bot-not-responding-in-group.md" "en" >}}).

### کیا بٹ مفت ہے؟
جی ہاں۔ رجسٹریشن یا ادائیگی کی ضرورت نہیں۔

### user ID اور chat_id میں کیا فرق ہے؟
**User ID** Telegram اکاؤنٹ کی شناخت کرتا ہے۔ **chat_id** بات چیت کی (پرائیویٹ چیٹ، گروپ، سپرگروپ، چینل)۔ دیکھیں [Telegram ID بمقابلہ username]({{< relref "telegram-id-vs-username.md" "en" >}}).
