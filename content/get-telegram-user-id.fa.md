**Telegram user ID** شناسه عددی حساب شماست. در اپ تلگرام نمایش داده نمی‌شود، اما ربات‌ها و یکپارچه‌سازی‌ها هنگام تعامل با آن‌ها می‌توانند آن را بخوانند. در ادامه روش‌های امن برای دریافت آن.

## روش ۱: استفاده از @print_id_bot (سریع‌ترین)

۱. [@print_id_bot](https://t.me/print_id_bot) را در تلگرام باز کنید.
۲. **شروع** را بزنید.
۳. ربات با **User ID** و **Chat ID** شما پاسخ می‌دهد (در چت خصوصی یکی هستند).
۴. روی عدد بزنید تا کپی شود.

ربات بر اساس تنظیم زبان تلگرام شما به ۱۶ زبان پاسخ می‌دهد. زبان‌های پشتیبانی‌نشده به انگلیسی برمی‌گردند.

## روش ۲: هر پیامی به ربات بفرستید

در چت خصوصی با @print_id_bot هر پیام، عکس، استیکر یا سند بفرستید. ربات با User ID و Chat ID شما پاسخ می‌دهد. می‌توانید از `/id` هم برای پاسخ کوتاه استفاده کنید.

## روش ۳: فوروارد پیام (و محدودیت‌ها)

اگر پیامی از کاربر دیگری به @print_id_bot فوروارد کنید، ربات ممکن است User ID فرستنده اصلی را نشان دهد. اما اگر فرستنده تنظیمات حریم خصوصی داشته باشد که هنگام فوروارد شناسه را مخفی کند، «ID hidden» خواهید دید. نمی‌توانید با شماره تلفن یا username به‌طور قابل اعتماد شناسه دیگری را بگیرید. جزئیات: [پیام‌های فوروارد و شناسه‌های مخفی]({{< relref "telegram-forwarded-message-id.md" "en" >}}).

## کارهایی که نمی‌توانید انجام دهید

- **جستجو با شماره تلفن**: نمی‌توانید فقط با شماره تلفن Telegram user ID بگیرید. سرویس‌هایی که ادعا می‌کنند اغلب کلاهبرداری یا نقض حریم خصوصی هستند.
- **جستجو با username**: usernameها شناسه نیستند. کاربر باید با ربات یا اپ تعامل کند تا شناسه را بگیرد.
- **دیدن شناسه دیگران بدون تعامل**: فقط می‌توانید شناسه چت‌هایی که در آن هستید یا کاربرانی که پیام‌های دریافتی شما را فوروارد می‌کنند (و اجازه فوروارد می‌دهند) را بگیرید.

## سوالات متداول

### آیا Telegram user ID من خصوصی است؟
در رابط تلگرام نمایش داده نمی‌شود، اما ربات‌ها و اپ‌ها هنگام تعامل می‌توانند آن را بخوانند. با سرویس‌های نامعتبر به اشتراک نگذارید.

### آیا می‌توانم Telegram user ID خود را عوض کنم؟
خیر. یک بار اختصاص داده می‌شود و تغییر نمی‌کند.

### چرا ربات برای پیام‌های فوروارد «ID hidden» نشان می‌دهد؟
فرستنده تنظیمات حریم خصوصی را فعال کرده که هنگام فوروارد شناسه را مخفی می‌کند. ببینید [شناسه فرستنده فوروارد مخفی]({{< relref "telegram-privacy-forwarded-id-hidden.md" "en" >}}).

### استفاده از @print_id_bot امن است؟
بله. ربات فقط شناسه‌هایی را نشان می‌دهد که از قبل به آن‌ها دسترسی دارید (خودتان یا از چت‌هایی که در آن هستید). داده‌های شما را فراتر از آنچه تلگرام فراهم می‌کند ذخیره یا به اشتراک نمی‌گذارد.

### Telegram user ID چه شکلی است؟
مقدار عددی، معمولاً ۹–۱۰ رقم. مثال: `123456789`.
