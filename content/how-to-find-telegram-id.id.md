## Cara tercepat (1 detik)

Buka bot dan tekan **Mulai**. Bot akan mengembalikan:

- **Telegram user ID** Anda
- **Username** Anda (jika ada)
- (Di grup) **chat_id** grup atau supergrup

## Metode 1 — Gunakan bot Telegram (direkomendasikan)

1. Buka bot: https://t.me/print_id_bot
2. Ketuk **Mulai**
3. Salin ID Telegram Anda dari pesan

**Mengapa metode ini terbaik:** sederhana, berfungsi di ponsel dan desktop, dan tidak memerlukan alat pengembang.

## Metode 2 — Dapatkan chat_id grup (untuk Bot API)

Jika Anda memerlukan **chat_id** grup untuk integrasi:

1. Tambahkan bot ke grup Anda
2. Kirim pesan apa pun di grup (atau gunakan perintah bot seperti `/id`)
3. Bot akan membalas dengan **chat_id** grup

> Catatan: ID grup dan supergrup sering kali berupa angka negatif. Itu normal.

## Masalah umum

- **Bot tidak membalas di grup saya:** pastikan bot memiliki izin untuk membaca pesan, dan periksa pengaturan mode privasi (tergantung cara bot dibuat).
- **Saya hanya melihat username, bukan ID:** username bukan ID. ID bersifat numerik.
- **Saya ingin ID orang lain:** alat ini untuk mendapatkan ID Anda sendiri atau chat_id obrolan tempat Anda berpartisipasi.

## FAQ

### Apakah ID Telegram saya bersifat pribadi?
ID Telegram Anda tidak ditampilkan secara publik di antarmuka Telegram, tetapi bot dan aplikasi dapat membacanya saat Anda berinteraksi dengan mereka.

### Bisakah saya mendapatkan ID melalui nomor telepon atau username?
Secara resmi, Anda umumnya tidak dapat memperoleh ID numerik seseorang secara andal hanya dari nomor telepon atau username tanpa interaksi. Hindari layanan yang menjanjikan «pencarian ID berdasarkan telepon» — sering kali penipuan atau pelanggaran privasi.

### Apa perbedaan antara user ID dan chat_id?
- **User ID** mengidentifikasi akun Telegram.
- **chat_id** mengidentifikasi percakapan (obrolan pribadi, grup, supergrup, saluran).
