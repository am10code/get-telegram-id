**የTelegram user ID** ዎ ለመለያዎ የቁጥር መለያ ነው። ቋሚ ነው፣ አንድ ጊዜ ይመደባል እና በTelegram መተግበሪያ ውስጥ በየትኛውም ቦታ አይታይም። ቦቶች እና ውህዶች ከእነሱ ጋር ሲገናኙ ብቻ ሊያንብቡት ይችላሉ — ለምሳሌ ቦት ሲከፍቱ እና ጀምር ሲጫኑ ወይም መልእክት ሲላኩ። ይህ ማእከል የuser ID ዎን ደህን በሆነ ሁኔታ ለማግኘት፣ ለመቅዳት እና ለመረዳት የሚረዱዎ መመሪያዎችን ይሰበስባል።

የሌላ ሰው መለያ በስልክ ቁጥር ወይም username ብቻ ማግኘት አይችሉም። «በስልክ መለያ ፍለጋ» የሚቀበሩ አገልግሎቶች ብዙውን ጊዜ ማጭበርበር ወይም የግላዊነት ማጣስራት ናቸው። መለያዎች በግንኙነት ብቻ ይገኛሉ፡ ተጠቃሚ ቦት ሲከፍት፣ መልእክት ሲያስተላልፍ (ለማስተላለፍ ቢፈቅድ) ወይም የሚደርስበትን ቻት ሲሳተፍ።

## መመሪያዎች

- **[የTelegram ID ዎን እንዴት ያግኙ]({{< relref "how-to-find-telegram-id.md" >}})** — የuser ID ዎን ለማግኘት ደረጃ በደረጃ ዘዴዎች። ፈጣኑ መንገድ ቦት መጠቀም ነው። ይክፈቱ፣ ጀምር ይጫኑ እና ቁጥሩን ይቅዱ።

- **[የTelegram user ID ያግኙ (ደህን ዘዴዎች)]({{< relref "get-telegram-user-id.md" >}})** — መለያዎን ለማግኘት ደህን መንገዶች፡ በቦት፣ መልእክት በማስረከብ ወይም በማስተላለፍ (ለግላዊነት ገደቦች ጋር)። ምን ማድረግ አይችሉም እና ለምን።

- **[የTelegram ID እና username፡ ለእንዴት ይለያሉ?]({{< relref "telegram-id-vs-username.md" >}})** — መለያዎች ቁጥራዊ እና ቋሚ ናቸው፤ usernames በመርጫ እና የሚቀየራቸው ናቸው። ለBot API ወይም ውህዶች መለያ መቼ ያስፈልጋል።

- **[የTelegram ID ዎን ደህን በሆነ ሁኔታ እንዴት ይቅዱ እና ይጋሩ]({{< relref "how-to-copy-telegram-id.md" >}})** — ከቦት ለመቅዳት ይጫኑ፣ ደህን በሆነ ሁኔታ ይጋሩ እና ማጭበርበር ያስወግዱ። መለያዎን ሲጋሩ የግላዊነት ምክሮች።

- **[Telegram ለምን የቁጥር መለያዎን አይያዝም (እና ምን ማድረግ እንዳለብዎ)]({{< relref "telegram-id-not-in-ui.md" >}})** — Telegram የቁጥር መለያዎን በመተግበሪያው ውስጥ ይደብቃል። ለምን የተሰወረ እና የሚያስፈልግዎ ሲሆን እንዴት ያግኙ።

- **[በTelegram ውስጥ የቀረቡ መልእክቶች፡ መለያዎች ለምን ሊሰወሩ ይችላሉ]({{< relref "telegram-forwarded-message-id.md" >}})** — መልእክት ወደ ቦት ሲያስተላልፉ፣ የመላኪያ መለያ ሊታይ ይችላል ወይም የግላዊነት ቅንብሮች ማስተላለፍ ቢያግዱ «ID hidden»።

## FAQ

### የTelegram ID ዎ ግላዊ ነው?

የTelegram ID ዎ በTelegram UI ውስጥ በይፋ አይታይም፣ ነገር ግን ቦቶች እና መተግበሪያዎች ከእነሱ ጋር ሲገናኙ ሊያንብቡት ይችላሉ። ቦቶች እና መተግበሪያዎች ከእነሱ ጋር ሲገናኙ ሊያንብቡት ይችላሉ። የትኛውን ቦቶች እና መተግበሪያዎች እንደሚጠቀሙ በመምረጥ ማን እንደሚያገኝ ይቆጣጠራሉ።

### የሌላ ሰው መለያ በስልክ ቁጥር ወይም username ማግኘት እችላለሁ?

አይ። የTelegram user ID በስልክ ቁጥር ወይም username ብቻ በታማኝነት ማግኘት አይችሉም። መለያዎች ተጠቃሚ ከቦት ወይም መተግበሪያ ጋር ሲገናኝ ወይም መልእክት ሲያስተላልፍ (እና ማስተላለፍ ሲፈቅድ) ብቻ ይገኛሉ። ያልተለያዩ የሚናገሩ አገልግሎቶችን ያስወግዱ።

### በuser ID እና chat_id መካከል ምን ልዩነት አለ?

**User ID** የTelegram መለያ ያሳያል። **chat_id** ውይይትን ያሳያል (ግል ቻት፣ ቡድን፣ ሱፐርቡድን፣ ሰርግ)። በግል ቻት ውስጥ የuser ID እና chat_id ተመሳሳይ ቁጥር ናቸው።
