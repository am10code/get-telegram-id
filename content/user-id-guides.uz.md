Ushbu hub sizga user ID ni xavfsiz topish, nusxalash va tushunishga yordam beradigan qoʻllanmalarni toʻplaydi.

## FAQ

### Mening Telegram ID im maxfiy mi?

Telegram ID ingiz Telegram interfeysida ochiq koʻrsatilmaydi, lekin botlar va ilovalar ular bilan oʻzaro taʼsir qilganingizda oʻqiy oladi. Qaysi botlar va ilovalardan foydalanishni tanlash orqali kim uni olishini nazorat qilasiz.

### Boshqa odamning ID sini telefon raqami yoki foydalanuvchi nomi orqali olish mumkinmi?

Yoʻq. Telegram user ID ni faqat telefon raqami yoki foydalanuvchi nomidan ishonchli olish mumkin emas. ID lar faqat foydalanuvchi bot yoki ilova bilan oʻzaro taʼsir qilganda yoki xabar yuborganida (va yuborishga ruxsat berilganda) olinadi. Boshqacha daʼvo qiladigan xizmatlardan saqlaning.

### user ID va chat_id oʻrtasidagi farq nima?

**User ID** Telegram hisobini aniqlaydi. **chat_id** suhbatni aniqlaydi (shaxsiy chat, guruh, superguruh, kanal). Shaxsiy chatda user ID va chat_id bir xil raqam.
