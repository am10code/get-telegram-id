## 最快方式（1 秒）

打开机器人并点击**开始**。它将返回：

- 您的 **Telegram 用户 ID**
- 您的 **用户名**（如有）
- （在群组中）群组或超级群组的 **chat_id**

## 方法 1 — 使用 Telegram 机器人（推荐）

1. 打开机器人：https://t.me/print_id_bot
2. 点击**开始**
3. 从消息中复制您的 Telegram ID

**此方法最佳的原因：** 简单，在手机和电脑上均可使用，无需开发者工具。

## 方法 2 — 获取群组 chat_id（用于 Bot API）

如需用于集成的群组 **chat_id**：

1. 将机器人添加到您的群组
2. 在群组中发送任意消息（或使用机器人的命令如 `/id`）
3. 机器人将回复群组的 **chat_id**

> 注意：群组和超级群组的 ID 通常为负数。这是正常的。

## 常见问题

- **机器人在我的群组中不回复：** 确保机器人有读取消息的权限，并检查隐私模式设置（取决于机器人的构建方式）。
- **我只看到用户名，没有 ID：** 用户名不是 ID。ID 是数字。
- **我想获取他人的 ID：** 此工具用于获取您自己的 ID 或您参与的聊天的 chat_id。

## 常见问题解答

### 我的 Telegram ID 是私密的吗？
您的 Telegram ID 不会在 Telegram 界面中公开显示，但机器人和应用在您与其交互时可以读取。

### 能否通过电话号码或用户名获取 ID？
官方而言，通常无法仅凭电话号码或用户名在无交互的情况下可靠地获取某人的数字 ID。避免使用承诺「通过电话查找 ID」的服务——它们往往是诈骗或侵犯隐私。

### user ID 和 chat_id 有什么区别？
- **User ID** 标识 Telegram 账户。
- **chat_id** 标识对话（私聊、群组、超级群组、频道）。
