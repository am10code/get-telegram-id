Боты и интеграции могут его прочитать только при взаимодействии с вами — например, когда вы откроете бота и нажмёте «Старт» или отправите сообщение. Получить чужой ID только по номеру телефона или username нельзя. Сервисы, обещающие «поиск ID по телефону», часто мошеннические или нарушают приватность. ## FAQ

### Приватен ли мой Telegram ID?

Ваш Telegram ID не отображается публично в интерфейсе Telegram, но боты и приложения могут его прочитать при взаимодействии с ними. Вы контролируете, кто его получит, выбирая, какими ботами и приложениями пользуетесь.

### Можно ли получить чужой ID по номеру телефона или username?

Нет. Надёжно получить Telegram user ID только по номеру телефона или username нельзя. ID получают только при взаимодействии пользователя с ботом или приложением, либо при пересылке сообщения (если пересылка разрешена). Избегайте сервисов, утверждающих обратное.

### В чём разница между user ID и chat_id?

**User ID** идентифицирует аккаунт Telegram. **chat_id** идентифицирует разговор (личный чат, группа, супергруппа, канал). В личном чате user ID и chat_id — одно и то же число.
