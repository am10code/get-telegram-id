## Самый быстрый способ (1 секунда)

Откройте бота и нажмите **Старт**. Он вернёт:

- Ваш **Telegram user ID**
- Ваш **username** (если он есть)
- (В группах) **chat_id** группы или супергруппы

## Способ 1 — Использовать Telegram-бота (рекомендуется)

1. Откройте бота: https://t.me/print_id_bot
2. Нажмите **Старт**
3. Скопируйте свой Telegram ID из сообщения

**Почему этот способ лучший:** он простой, работает на мобильных и десктопе, не требует инструментов разработчика.

## Способ 2 — Получить chat_id группы (для Bot API)

Если вам нужен **chat_id** группы для интеграций:

1. Добавьте бота в свою группу
2. Отправьте любое сообщение в группе (или используйте команду бота, например `/id`)
3. Бот ответит **chat_id** группы

> Примечание: ID групп и супергрупп часто являются отрицательными числами. Это нормально.

## Частые проблемы

- **Бот не отвечает в моей группе:** убедитесь, что у бота есть разрешение на чтение сообщений, и проверьте настройки режима конфиденциальности (в зависимости от того, как построен бот).
- **Я вижу только username, а не ID:** username — это не ID. ID — числовые.
- **Мне нужен ID другого человека:** этот инструмент предназначен для получения своего ID или chat_id чатов, в которых вы участвуете.

## FAQ

### Является ли мой Telegram ID приватным?
Ваш Telegram ID не отображается публично в интерфейсе Telegram, но боты и приложения могут его прочитать при взаимодействии с ними.

### Можно ли получить ID по номеру телефона или username?
Официально надёжно получить числовой ID человека только по номеру телефона или username без взаимодействия обычно нельзя. Избегайте сервисов, обещающих «поиск ID по телефону» — это часто мошенничество или нарушение конфиденциальности.

### В чём разница между user ID и chat_id?
- **User ID** идентифицирует аккаунт Telegram.
- **chat_id** идентифицирует разговор (личный чат, группа, супергруппа, канал).
