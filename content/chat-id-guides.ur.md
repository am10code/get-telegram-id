## FAQ

### میری گروپ کا chat_id کیوں منفی ہے؟

گروپ اور سپرگروپ chat_ids Telegram Bot API میں عام طور پر منفی ہوتے ہیں۔ یہ user IDs (پرائیویٹ چیٹ میں مثبت) سے ممتاز کرنے کے لیے ڈیزائن کے مطابق ہے۔ معمول اور متوقع۔

### اگر بٹ میری گروپ میں جواب نہ دے تو کیا کریں؟

گروپس میں بٹس اکثر صرف اس وقت جواب دیتے ہیں جب آپ انہیں جواب دیں، `/id` یا `/help` بھیجیں، یا منشن کریں (مثلاً @print_id_bot)۔ اگر اب بھی خاموش ہے تو چیک کریں کہ بٹ کو پیغامات پڑھنے کی اجازت ہے اور پرائیویسی موڈ (اگر لاگو ہو) گروپ پیغامات دیکھنے کی اجازت دیتا ہے۔

### سپرگروپ میں اپ گریڈ ہونے پر chat_id بدلتا ہے؟

جی ہاں۔ جب گروپ سپرگروپ میں اپ گریڈ ہوتا ہے، chat_id بدل سکتا ہے۔ اگر آپ نے پرانا گروپ chat_id محفوظ کیا ہے تو شاید اب کام نہ کرے۔ بٹ کو دوبارہ شامل کریں اور خوش آمدید پیغام یا `/id` سے نیا chat_id حاصل کریں۔
