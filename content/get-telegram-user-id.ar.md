**Telegram user ID** هو معرف رقمي لحسابك. لا يُعرض في تطبيق تلغرام، لكن البوتات والتكاملات يمكنها قراءته عند التفاعل معها. إليك طرق آمنة للحصول عليه.

## الطريقة 1: استخدم @print_id_bot (الأسرع)

1. افتح [@print_id_bot](https://t.me/print_id_bot) في تلغرام.
2. اضغط **ابدأ**.
3. يرد البوت بـ **User ID** و **Chat ID** (متطابقان في الدردشة الخاصة).
4. اضغط على الرقم للنسخ.

يستجيب البوت بـ 16 لغة حسب إعداد لغة تلغرام. اللغات غير المدعومة تعود للإنجليزية.

## الطريقة 2: أرسل أي رسالة للبوت

في الدردشة الخاصة مع @print_id_bot، أرسل أي رسالة أو صورة أو ملصق أو مستند. يرد البوت بـ User ID و Chat ID. يمكنك أيضاً استخدام `/id` لرد قصير.

## الطريقة 3: إعادة توجيه رسالة (والقيود)

إذا أعدت توجيه رسالة من مستخدم آخر إلى @print_id_bot، قد يعرض البوت User ID المرسل الأصلي. لكن إذا كان للمرسل إعدادات خصوصية تخفي معرفه عند إعادة التوجيه، فسترى "ID hidden". لا يمكن البحث عن معرف شخص آخر بموثوقية عبر رقم الهاتف أو username. انظر [الرسائل المُعاد توجيهها والمعرفات المخفية]({{< relref "telegram-forwarded-message-id.md" "en" >}}) للتفاصيل.

## ما لا يمكنك فعله

- **البحث برقم الهاتف**: لا يمكن الحصول على Telegram user ID من رقم الهاتف وحده. الخدمات التي تدّعي ذلك غالباً احتيال أو تنتهك الخصوصية.
- **البحث بـ username**: الـ usernames ليست معرفات. يحتاج المستخدم للتفاعل مع بوت أو تطبيق للحصول على معرفه.
- **رؤية معرفات الآخرين دون تفاعل**: يمكنك فقط الحصول على معرفات الدردشات التي أنت فيها أو من المستخدمين الذين يعيدون توجيه رسائل تستلمها (والذين يسمحون بإعادة التوجيه).

## الأسئلة الشائعة

### هل Telegram user ID خاصتي سري؟
لا يُعرض في واجهة تلغرام، لكن البوتات والتطبيقات يمكنها قراءته عند التفاعل. لا تشاركه مع خدمات غير موثوقة.

### هل يمكنني تغيير Telegram user ID؟
لا. يُعيّن مرة واحدة ولا يتغير.

### لماذا يعرض البوت "ID hidden" للرسائل المُعاد توجيهها؟
أطلق المرسل إعدادات خصوصية تخفي معرفه عند إعادة التوجيه. انظر [معرف المرسل المُعاد توجيهه مخفي]({{< relref "telegram-privacy-forwarded-id-hidden.md" "en" >}}).

### هل استخدام @print_id_bot آمن؟
نعم. يعرض البوت فقط المعرفات التي لديك صلاحية الوصول إليها (خاصتك أو من الدردشات التي أنت فيها). لا يخزن أو يشارك بياناتك بما يتجاوز ما يوفره تلغرام.

### كيف يبدو Telegram user ID؟
قيمة رقمية، غالباً 9–10 أرقام. مثال: `123456789`.
