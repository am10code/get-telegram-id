**Telegram user ID** ዎ ለመለያዎ የቁጥር መለያ ነው። በቴሌግራም መተግበሪያ ውስጥ አይታይም፣ ግን ቦቶች እና ውህዶች ያለዎት ጉዳይ ሲገናኙ ሊያነቡት ይችላሉ። ይህንን በደህንነት ለማግኘት ዘዴዎች እዚህ አሉ።

## ዘዴ 1 ፡ @print_id_bot ይጠቀሙ (ፈጣኑ)

1. በቴሌግራም ውስጥ [@print_id_bot](https://t.me/print_id_bot) ይክፈቱ።
2. **ጀምር** ይጫኑ።
3. ቦቱ በሁለትዎ **User ID** እና **Chat ID** ይመልሳል (በግል ቻት ውስጥ ማጣቀሻ ናቸው)።
4. ለመቅዳት በቁጥሩ ላይ ይጫኑ።

ቦቱ በቴሌግራም የቋንቋ ቅንብርዎ መሰረት በ16 ቋንቋዎች ይመልሳል። ያልተደገፉ ቋንቋዎች ወደ እንግሊዝኛ ይመለሳሉ።

## ዘዴ 2 ፡ ለቦቱ ማንኛውንም መልእክት ይላኩ

## ዘዴ 3 ፡ መልእክት ማስተላለፍ (እና ገደቦች)

## ማድረግ የማይችሉት

- **በስልክ ቁጥር መፈለግ**: በስልክ ቁጥር ብቻ የTelegram user ID ማግኘት አይችሉም። ይህንን የሚያስደስቱ አገልግሎቶች ብዙውን ጊዜ የማጭበርበር ወይም የግል ቅንብር ማስፋፋት ናቸው።
- **በusername መፈለግ**: Usernameዎች ID አይደሉም። መለያ ለማግኘት ተጠቃሚ ከቦት ወይም መተግበሪያ ጋር መስራት አለበት።
- **ያለስራት የሌሎች መለያዎችን ማየት**: ያሉበትን ቻቶች ወይም የሚያገኙትን መልእክቶች የሚያስተላልፉ ተጠቃሚዎችን (እና ማስተላለፍ የሚፈቅዱ) መለያዎችን ብቻ ማግኘት ይችላሉ።
