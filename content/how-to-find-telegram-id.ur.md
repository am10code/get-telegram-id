## سب سے تیز طریقہ (1 سیکنڈ)

بوٹ کھولیں اور **شروع کریں** دبائیں۔ یہ واپس دے گا:

- آپ کا **Telegram user ID**
- آپ کا **username** (اگر ہے تو)
- (گروپس میں) گروپ یا سپرگروپ کا **chat_id**

## طریقہ 1 — Telegram بوٹ استعمال کریں (تجویز کردہ)

1. بوٹ کھولیں: https://t.me/print_id_bot
2. **شروع کریں** پر ٹیپ کریں
3. پیغام سے اپنا Telegram ID کاپی کریں

**یہ طریقہ بہترین کیوں:** آسان ہے، موبائل اور ڈیسک ٹاپ دونوں پر کام کرتا ہے، اور ڈویلپر ٹولز کی ضرورت نہیں۔

## طریقہ 2 — گروپ chat_id حاصل کریں (Bot API کے لیے)

اگر آپ کو انٹیگریشنز کے لیے گروپ **chat_id** چاہیے:

1. بوٹ کو اپنے گروپ میں شامل کریں
2. گروپ میں کوئی پیغام بھیجیں (یا بوٹ کے کمانڈ جیسے `/id` استعمال کریں)
3. بوٹ گروپ کے **chat_id** کے ساتھ جواب دے گا

> نوٹ: گروپ اور سپرگروپ ID اکثر منفی نمبر ہوتے ہیں۔ یہ عام بات ہے۔

## عام مسائل

- **بوٹ میرے گروپ میں جواب نہیں دیتا:** یقینی بنائیں کہ بوٹ کو پیغامات پڑھنے کی اجازت ہے، اور پرائیویسی موڈ سیٹنگز چیک کریں (بوٹ کیسے بنایا گیا ہے اس پر منحصر)۔
- **مجھے صرف username نظر آتا ہے، ID نہیں:** usernames ID نہیں ہیں۔ ID عددی ہیں۔
- **مجھے کسی اور کا ID چاہیے:** یہ ٹول اپنا ID یا ان چیٹس کا chat_id حاصل کرنے کے لیے ہے جن میں آپ شامل ہیں۔

## FAQ

### کیا میرا Telegram ID پرائیویٹ ہے؟
آپ کا Telegram ID Telegram UI میں عوامی طور پر نہیں دکھایا جاتا، لیکن بوٹس اور ایپس اسے پڑھ سکتے ہیں جب آپ ان کے ساتھ تعامل کرتے ہیں۔

### کیا میں فون نمبر یا username سے ID حاصل کر سکتا ہوں؟
سرکاری طور پر، عام طور پر بغیر تعامل کے صرف فون نمبر یا username سے کسی کا عددی ID قابل اعتماد طریقے سے حاصل نہیں کر سکتے۔ ان سروسز سے بچیں جو «فون سے ID تلاش» کا وعدہ کرتی ہیں — یہ اکثر دھوکہ دہی یا پرائیویسی کی خلاف ورزی ہوتی ہیں۔

### user ID اور chat_id میں کیا فرق ہے؟
- **User ID** Telegram اکاؤنٹ کی شناخت کرتا ہے۔
- **chat_id** بات چیت کی شناخت کرتا ہے (پرائیویٹ چیٹ، گروپ، سپرگروپ، چینل)۔
