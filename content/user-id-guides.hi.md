आपका **Telegram user ID** आपके खाते के लिए एक संख्यात्मक पहचानकर्ता है। यह स्थिर है, एक बार निर्दिष्ट होता है और Telegram ऐप में कहीं भी दिखाई नहीं देता। बॉट और इंटीग्रेशन इसे तभी पढ़ सकते हैं जब आप उनके साथ इंटरैक्ट करते हैं — उदाहरण के लिए, जब आप बॉट खोलते हैं और स्टार्ट टैप करते हैं, या संदेश भेजते हैं। यह हब आपको अपना user ID सुरक्षित रूप से खोजने, कॉपी करने और समझने में मदद करने के लिए गाइड एकत्र करता है।

आप किसी और का ID केवल फोन नंबर या यूजरनेम से नहीं प्राप्त कर सकते। «फोन से ID खोज» का वादा करने वाली सेवाएं अक्सर घोटाले या गोपनीयता उल्लंघन होती हैं। ID केवल इंटरैक्शन के माध्यम से प्राप्त होते हैं: जब उपयोगकर्ता बॉट खोलता है, संदेश फॉरवर्ड करता है (अगर अनुमति दे), या उस चैट में भाग लेता है जिस तक आपकी पहुंच है।

## FAQ

### क्या मेरा Telegram ID निजी है?

आपका Telegram ID Telegram UI में सार्वजनिक रूप से नहीं दिखाया जाता, लेकिन बॉट और ऐप इंटरैक्ट करने पर पढ़ सकते हैं। आप कौन इसे प्राप्त करता है यह चुनकर नियंत्रित करते हैं कि किन बॉट और ऐप का उपयोग करते हैं।

### क्या मैं फोन नंबर या यूजरनेम से किसी और का ID प्राप्त कर सकता हूं?

नहीं। आप केवल फोन नंबर या यूजरनेम से विश्वसनीय रूप से Telegram user ID प्राप्त नहीं कर सकते। ID केवल तभी प्राप्त होते हैं जब उपयोगकर्ता बॉट या ऐप के साथ इंटरैक्ट करता है, या संदेश फॉरवर्ड करता है (और फॉरवर्डिंग की अनुमति देता है)। अन्यथा दावा करने वाली सेवाओं से बचें।

### user ID और chat_id में क्या अंतर है?

**User ID** Telegram खाते की पहचान करता है। **chat_id** बातचीत की पहचान करता है (निजी चैट, ग्रुप, सुपरग्रुप, चैनल)। निजी चैट में, user ID और chat_id एक ही नंबर हैं।
