**Telegram user ID** — это числовой идентификатор вашего аккаунта. Он не отображается в приложении Telegram, но боты и интеграции могут его читать при взаимодействии с вами. Ниже — безопасные способы его получить.

## Способ 1: Использовать @print_id_bot (самый быстрый)

1. Откройте [@print_id_bot](https://t.me/print_id_bot) в Telegram.
2. Нажмите **Старт**.
3. Бот ответит вашим **User ID** и **Chat ID** (в личке они совпадают).
4. Нажмите на число, чтобы скопировать.

Бот отвечает на 16 языках в зависимости от настроек Telegram. Неподдерживаемые языки переключаются на английский.

## Способ 2: Отправить любое сообщение боту

В личке с @print_id_bot отправьте любое сообщение, фото, стикер или документ. Бот ответит вашим User ID и Chat ID. Можно также использовать `/id` для краткого ответа.

## Способ 3: Переслать сообщение (и ограничения)

Если вы перешлёте сообщение от другого пользователя в @print_id_bot, бот может показать User ID исходного отправителя. Однако если у отправителя включены настройки приватности, скрывающие ID при пересылке, вы увидите «ID hidden». Нельзя надёжно получить чужой ID по номеру телефона или username. Подробнее: [пересланные сообщения и скрытые ID]({{< relref "telegram-forwarded-message-id.md" "en" >}}).

## Чего нельзя делать

- **Искать по номеру телефона**: Нельзя получить Telegram user ID только по номеру телефона. Сервисы, которые это обещают, часто мошеннические или нарушают приватность.
- **Искать по username**: Username — это не ID. Нужно, чтобы пользователь взаимодействовал с ботом или приложением.
- **Видеть чужие ID без взаимодействия**: Можно получить только ID чатов, в которых вы участвуете, или пользователей, переславших вам сообщения (и разрешивших пересылку).

## FAQ

### Приватен ли мой Telegram user ID?
Он не отображается в интерфейсе Telegram, но боты и приложения могут его читать при взаимодействии. Не передавайте его ненадёжным сервисам.

### Можно ли изменить свой Telegram user ID?
Нет. Он назначается один раз и не меняется.

### Почему бот показывает «ID hidden» для пересланных сообщений?
Отправитель включил настройки приватности, скрывающие ID при пересылке. См. [скрытый ID отправителя пересланного сообщения]({{< relref "telegram-privacy-forwarded-id-hidden.md" "en" >}}).

### Безопасно ли использовать @print_id_bot?
Да. Бот показывает только те ID, к которым у вас уже есть доступ (ваш или из чатов, где вы участвуете). Он не хранит и не передаёт ваши данные сверх того, что предоставляет Telegram.

### Как выглядит Telegram user ID?
Это числовое значение, обычно 9–10 цифр. Пример: `123456789`.
