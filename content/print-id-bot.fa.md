@print_id_bot یک ربات تلگرام است که **Telegram user ID**، **chat_id** (برای گروه‌ها و سوپرگروه‌ها) و شناسه کانال شما را نمایش می‌دهد. در چت خصوصی و گروه‌ها کار می‌کند و بر اساس تنظیم زبان تلگرام شما به ۱۶ زبان پاسخ می‌دهد.

## دستورات

| دستور | عملکرد |
|-------|--------|
| `/start` | پیام خوش‌آمد، User ID و Chat ID شما، دکمه‌های اینلاین برای /id و /help |
| `/id` | User ID و Chat ID شما (پاسخ کوتاه) |
| `/help` | متن راهنما و لیست دستورات |
| `/json` | پیام فعلی به صورت JSON (برای توسعه‌دهندگان) |
| `/dump` | تجزیه کامل آپدیت و JSON خام (فقط در چت خصوصی) |

در چت خصوصی، ارسال هر پیامی (متن، عکس، استیکر و غیره) نیز User ID و Chat ID شما را برمی‌گرداند. برای پیام‌های فوروارد شده، ربات شناسه نویسنده را در صورت امکان نمایش می‌دهد، یا «ID hidden» اگر تنظیمات حریم خصوصی فرستنده آن را مسدود کند. جزئیات: [پیام‌های فوروارد و شناسه‌های مخفی]({{< relref "telegram-forwarded-message-id.md" "en" >}}).

## زبان‌های پشتیبانی‌شده (۱۶)

ربات زبان پاسخ را از `language_code` تلگرام شما انتخاب می‌کند. پشتیبانی‌شده: انگلیسی، روسی، فارسی، ازبکی، هندی، چینی، ترکی، پرتغالی، اسپانیایی، عربی، اندونزیایی، آلمانی، اردو، فرانسوی، تاگالوگ، امهری. کدهای پشتیبانی‌نشده به انگلیسی برمی‌گردند. ببینید [نحوه کار انتخاب زبان]({{< relref "telegram-id-bot-languages.md" "en" >}}).

## رفتار در گروه و سوپرگروه

در گروه‌ها ربات فقط وقتی پاسخ می‌دهد که:

- به پیام ربات پاسخ دهید
- `/id` یا `/help` بفرستید (با یا بدون @print_id_bot)
- هر دستوری با @print_id_bot بفرستید (مثلاً `/foo@print_id_bot`)
- @print_id_bot را در پیام خود ذکر کنید

در غیر این صورت ساکت می‌ماند. هنگام اضافه شدن به گروه، پیام خوش‌آمد با نام گروه و chat_id می‌فرستد. راهنمای کوتاه: [افزودن ربات به گروه برای دریافت chat_id]({{< relref "add-bot-to-group-chat-id.md" "en" >}}).

## امکانات توسعه‌دهنده

- **/json** — پیام فعلی را به صورت JSON برمی‌گرداند.
- **/dump** — تجزیه کامل آپدیت و JSON خام (فقط چت خصوصی). برای دیباگ یکپارچه‌سازی Bot API مفید است. ببینید [بررسی JSON آپدیت تلگرام]({{< relref "telegram-json-update.md" "en" >}}).

## سوالات متداول

### آیا ربات در کانال‌ها کار می‌کند؟
ربات در چت خصوصی و گروه‌ها/سوپرگروه‌ها کار می‌کند. برای کانال‌ها شناسه کانال از جریان دیگری لازم است (مثلاً ارسال به عنوان کانال و استفاده از رباتی که آپدیت دریافت می‌کند). ببینید [دریافت شناسه کانال تلگرام برای Bot API]({{< relref "how-to-get-channel-id-bot-api.md" "en" >}}).

### آیا می‌توانم Telegram user ID شخص دیگری را بگیرم؟
ربات شناسه خودتان یا chat_id چت‌هایی که در آن هستید را نشان می‌دهد. برای پیام‌های فوروارد، شناسه فرستنده را فقط اگر اجازه فوروارد بدهند نشان می‌دهد؛ وگرنه «ID hidden». نمی‌توان با شماره تلفن یا username شناسه را جستجو کرد. ببینید [Telegram ID در مقابل username]({{< relref "telegram-id-vs-username.md" "en" >}}).

### چرا ربات در گروه من پاسخ نمی‌دهد؟
ربات فقط وقتی پاسخ می‌دهد که به آن پاسخ دهید، /id یا /help بفرستید، یا @print_id_bot را ذکر کنید. ببینید [ربات در گروه پاسخ نمی‌دهد]({{< relref "bot-not-responding-in-group.md" "en" >}}) برای چک‌لیست کامل.

### آیا ربات رایگان است؟
بله. ثبت‌نام یا پرداخت لازم نیست.

### تفاوت user ID و chat_id چیست؟
**User ID** یک حساب تلگرام را شناسایی می‌کند. **chat_id** یک مکالمه را (چت خصوصی، گروه، سوپرگروه، کانال). ببینید [Telegram ID در مقابل username]({{< relref "telegram-id-vs-username.md" "en" >}}).
