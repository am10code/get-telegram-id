## پرائیویٹ چیٹ میں اپنا Telegram user ID کیسے حاصل کریں

@print_id_bot کو پرائیویٹ چیٹ میں کھولیں اور **شروع کریں** دبائیں۔ بوٹ فوری طور پر آپ کا **Telegram user ID**، username (اگر ہے تو) اور chat_id واپس دے گا۔ اور کچھ نہیں چاہیے: بس پیغام سے نمبر کاپی کریں۔

پرائیویٹ چیٹ میں بوٹ ہر پیغام کا جواب دیتا ہے: ٹیکسٹ، فوٹو، اسٹیکر یا `/id` کمانڈ۔ جو بھی بھیجیں، اپنا ID پائیں گے۔ پرائیویٹ ہے: جواب صرف آپ دیکھتے ہیں۔

## گروپ کا chat_id کیسے حاصل کریں

گروپ یا سپرگروپ کا **chat_id** حاصل کرنے کے لیے @print_id_bot کو گروپ میں شامل کریں۔ شامل ہونے کے بعد بوٹ گروپ نام اور chat_id کے ساتھ خوش آمدید پیغام بھیجے گا۔

گروپس میں بوٹ **صرف مخصوص اقدامات پر** جواب دیتا ہے۔ chat_id حاصل کرنے کے لیے ان میں سے ایک کریں:

- **/id** یا **/help** کمانڈ بھیجیں (@print_id_bot کے ساتھ یا بغیر)
- بوٹ کے پیغام کا جواب دیں
- اپنے پیغام میں **@print_id_bot** کا ذکر کریں
- @print_id_bot کے ساتھ کوئی بھی کمانڈ بھیجیں (مثلاً `/foo@print_id_bot`)

بوٹ کا ذکر کیے بغیر عام پیغامات جواب نہیں دیتے: اس طرح بوٹ اسپیم سے بچتا ہے۔ گروپ اور سپرگروپ ID اکثر منفی ہوتے ہیں؛ یہ عام بات ہے۔

## FAQ

### کیا میرا Telegram ID پرائیویٹ ہے؟

آپ کا Telegram ID ایپ میں عوامی طور پر نہیں دکھایا جاتا۔ بوٹس اور ایپس اسے صرف تب حاصل کرتے ہیں جب آپ ان کے ساتھ تعامل کرتے ہیں — مثلاً بوٹ کھول کر شروع کریں دبانا۔ آپ فیصلہ کرتے ہیں کہ کن بوٹس پر بھروسہ کرنا ہے۔

### بوٹ گروپ میں جواب کیوں نہیں دیتا؟

بوٹ صرف تب جواب دیتا ہے جب آپ اس کے پیغام کا جواب دیتے ہیں، /id یا /help بھیجتے ہیں، یا @print_id_bot کا ذکر کرتے ہیں۔ ان ٹرگرز کے بغیر عام پیغامات نظر انداز کرتا ہے۔ یقینی بنائیں کہ بوٹ گروپ میں ہے اور پیغامات پڑھ سکتا ہے۔

### user ID اور chat_id میں کیا فرق ہے؟

**user ID** آپ کے Telegram اکاؤنٹ کی شناخت کرتا ہے۔ **chat_id** چیٹ کی شناخت کرتا ہے: پرائیویٹ، گروپ، سپرگروپ یا چینل۔ پرائیویٹ چیٹ میں user ID اور chat_id ایک ہی نمبر ہوتے ہیں۔ گروپس کے لیے chat_id اکثر منفی ہوتا ہے۔
