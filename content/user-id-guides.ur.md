## FAQ

### کیا میرا Telegram ID پرائیویٹ ہے؟

آپ کا Telegram ID Telegram UI میں عوامی طور پر نہیں دکھایا جاتا، لیکن بوٹس اور ایپس تعامل کے وقت پڑھ سکتے ہیں۔ آپ کن بوٹس اور ایپس استعمال کرتے ہیں منتخب کر کے کنٹرول کرتے ہیں کہ کون اسے حاصل کرتا ہے۔

### کیا میں فون نمبر یا username سے کسی اور کا ID حاصل کر سکتا ہوں؟

نہیں۔ آپ صرف فون نمبر یا username سے قابل اعتماد Telegram user ID حاصل نہیں کر سکتے۔ ID صرف اس وقت حاصل ہوتے ہیں جب صارف بوٹ یا ایپ کے ساتھ تعامل کرتا ہے، یا پیغام فارورڈ کرتا ہے (اور فارورڈنگ کی اجازت دیتا ہے)۔ دوسری صورت میں دعویٰ کرنے والی سروسز سے بچیں۔

### user ID اور chat_id میں کیا فرق ہے؟

**User ID** Telegram اکاؤنٹ کی شناخت کرتا ہے۔ **chat_id** بات چیت کی شناخت کرتا ہے (پرائیویٹ چیٹ، گروپ، سپرگروپ، چینل)۔ پرائیویٹ چیٹ میں، user ID اور chat_id ایک ہی نمبر ہیں۔
