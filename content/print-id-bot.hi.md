@print_id_bot एक Telegram बॉट है जो आपका **Telegram user ID**, **chat_id** (ग्रुप और सुपरग्रुप के लिए) और चैनल ID दिखाता है। यह प्राइवेट चैट और ग्रुप में काम करता है, आपकी Telegram भाषा सेटिंग के अनुसार 16 भाषाओं में जवाब देता है।

## कमांड

| कमांड | कार्य |
|-------|------|
| `/start` | स्वागत संदेश, User ID और Chat ID, /id और /help के लिए इनलाइन बटन |
| `/id` | User ID और Chat ID (संक्षिप्त जवाब) |
| `/help` | पूर्ण सहायता पाठ और कमांड सूची |
| `/json` | वर्तमान संदेश JSON में (डेवलपर्स के लिए) |
| `/dump` | पूर्ण अपडेट विवरण और कच्चा JSON (केवल प्राइवेट चैट) |

प्राइवेट चैट में कोई भी संदेश (टेक्स्ट, फोटो, स्टिकर आदि) भेजने पर भी User ID और Chat ID मिलता है। फॉरवर्ड किए गए संदेशों के लिए बॉट लेखक का ID दिखाता है जब उपलब्ध हो, या "ID hidden" जब भेजने वाले की प्राइवेसी सेटिंग ब्लॉक करती है। विवरण: [फॉरवर्ड संदेश और छिपे ID]({{< relref "telegram-forwarded-message-id.md" "en" >}}).

## समर्थित भाषाएँ (16)

बॉट आपके Telegram `language_code` से जवाब की भाषा चुनता है। समर्थित: अंग्रेज़ी, रूसी, फारसी, उज़्बेक, हिंदी, चीनी, तुर्की, पुर्तगाली, स्पेनिश, अरबी, इंडोनेशियाई, जर्मन, उर्दू, फ्रेंच, तागालोग, अम्हारिक। असमर्थित कोड अंग्रेज़ी पर वापस जाते हैं। देखें [भाषा चयन कैसे काम करता है]({{< relref "telegram-id-bot-languages.md" "en" >}}).

## ग्रुप और सुपरग्रुप व्यवहार

ग्रुप में बॉट तभी जवाब देता है जब आप:

- बॉट के संदेश का जवाब देते हैं
- `/id` या `/help` भेजते हैं (@print_id_bot के साथ या बिना)
- @print_id_bot के साथ कोई कमांड भेजते हैं (जैसे `/foo@print_id_bot`)
- अपने संदेश में @print_id_bot का उल्लेख करते हैं

अन्यथा चुप रहता है। ग्रुप में जोड़े जाने पर ग्रुप नाम और chat_id के साथ स्वागत संदेश भेजता है। संक्षिप्त गाइड: [chat_id के लिए ग्रुप में बॉट जोड़ें]({{< relref "add-bot-to-group-chat-id.md" "en" >}}).

## डेवलपर सुविधाएँ

- **/json** — वर्तमान संदेश JSON में लौटाता है।
- **/dump** — पूर्ण अपडेट विवरण और कच्चा JSON (केवल प्राइवेट चैट)। Bot API एकीकरण डिबग करने के लिए उपयोगी। देखें [Telegram अपडेट JSON जांचना]({{< relref "telegram-json-update.md" "en" >}}).

## सामान्य प्रश्न

### क्या बॉट चैनल में काम करता है?
बॉट प्राइवेट चैट और ग्रुप/सुपरग्रुप में काम करता है। चैनल के लिए चैनल ID दूसरे फ्लो से चाहिए (जैसे चैनल के रूप में पोस्ट और अपडेट प्राप्त करने वाला बॉट)। देखें [Bot API के लिए Telegram चैनल ID प्राप्त करें]({{< relref "how-to-get-channel-id-bot-api.md" "en" >}}).

### क्या मैं किसी और का Telegram user ID प्राप्त कर सकता हूँ?
बॉट आपका अपना ID या जिन चैट में आप हैं उनका chat_id दिखाता है। फॉरवर्ड संदेशों के लिए भेजने वाले का ID तभी दिखाता है जब वे फॉरवर्ड की अनुमति दें; नहीं तो "ID hidden"। फोन नंबर या username से ID खोज नहीं सकते। देखें [Telegram ID बनाम username]({{< relref "telegram-id-vs-username.md" "en" >}}).

### बॉट मेरे ग्रुप में जवाब क्यों नहीं देता?
बॉट तभी जवाब देता है जब आप उसे जवाब देते हैं, /id या /help भेजते हैं, या @print_id_bot का उल्लेख करते हैं। पूर्ण चेकलिस्ट: [ग्रुप में जवाब न देने वाला बॉट]({{< relref "bot-not-responding-in-group.md" "en" >}}).

### क्या बॉट मुफ्त है?
हाँ। रजिस्ट्रेशन या भुगतान की जरूरत नहीं।

### user ID और chat_id में क्या अंतर है?
**User ID** Telegram अकाउंट की पहचान करता है। **chat_id** बातचीत की (प्राइवेट चैट, ग्रुप, सुपरग्रुप, चैनल)। देखें [Telegram ID बनाम username]({{< relref "telegram-id-vs-username.md" "en" >}}).
