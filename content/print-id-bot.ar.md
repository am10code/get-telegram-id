@print_id_bot هو بوت تلغرام يعرض **Telegram user ID** و**chat_id** (للمجموعات والمجموعات الفائقة) ومعرف القناة. يعمل في الدردشة الخاصة والمجموعات، مع ردود بـ 16 لغة حسب إعداد لغة تلغرام.

## الأوامر

| الأمر | الوظيفة |
|-------|---------|
| `/start` | رسالة ترحيب، User ID وChat ID، أزرار مضمنة لـ /id و/help |
| `/id` | User ID وChat ID (رد قصير) |
| `/help` | نص المساعدة الكامل وقائمة الأوامر |
| `/json` | الرسالة الحالية بصيغة JSON (للمطورين) |
| `/dump` | تفصيل كامل للتحديث وJSON خام (الدردشة الخاصة فقط) |

في الدردشة الخاصة، إرسال أي رسالة (نص، صورة، ملصق، إلخ.) يعيد أيضاً User ID وChat ID. للرسائل المُعاد توجيهها، يعرض البوت معرف المؤلف عند توفره، أو "ID hidden" عند حظر إعدادات الخصوصية للمرسل. انظر [الرسائل المُعاد توجيهها والمعرفات المخفية]({{< relref "telegram-forwarded-message-id.md" "en" >}}) للتفاصيل.

## اللغات المدعومة (16)

يختار البوت لغة الرد من `language_code` في تلغرام. المدعومة: الإنجليزية، الروسية، الفارسية، الأوزبكية، الهندية، الصينية، التركية، البرتغالية، الإسبانية، العربية، الإندونيسية، الألمانية، الأردية، الفرنسية، التاغالوغية، الأمهرية. الرموز غير المدعومة تعود للإنجليزية. انظر [كيف يعمل اختيار اللغة]({{< relref "telegram-id-bot-languages.md" "en" >}}).

## السلوك في المجموعات والمجموعات الفائقة

في المجموعات، يرد البوت فقط عندما:

- ترد على رسالة البوت
- ترسل `/id` أو `/help` (مع أو بدون @print_id_bot)
- ترسل أي أمر مع @print_id_bot (مثلاً `/foo@print_id_bot`)
- تذكر @print_id_bot في رسالتك

وإلا يبقى صامتاً. عند إضافته لمجموعة، يرسل رسالة ترحيب باسم المجموعة وchat_id. انظر [إضافة بوت للمجموعة للحصول على chat_id]({{< relref "add-bot-to-group-chat-id.md" "en" >}}) لدليل سريع.

## ميزات المطورين

- **/json** — يعيد الرسالة الحالية بصيغة JSON.
- **/dump** — تفصيل كامل للتحديث وJSON خام (الدردشة الخاصة فقط). مفيد لتصحيح تكاملات Bot API. انظر [فحص JSON تحديث تلغرام]({{< relref "telegram-json-update.md" "en" >}}).

## الأسئلة الشائعة

### هل يعمل البوت في القنوات؟
يعمل البوت في الدردشة الخاصة والمجموعات/المجموعات الفائقة. للقنوات تحتاج معرف القناة من تدفق آخر (مثلاً النشر كقناة واستخدام بوت يستقبل التحديثات). انظر [الحصول على معرف قناة تلغرام لـ Bot API]({{< relref "how-to-get-channel-id-bot-api.md" "en" >}}).

### هل يمكنني الحصول على Telegram user ID لشخص آخر؟
يعرض البوت معرفك أو chat_id الدردشات التي أنت فيها. للرسائل المُعاد توجيهها يعرض معرف المرسل فقط إذا سمح بإعادة التوجيه؛ وإلا "ID hidden". لا يمكن البحث عن معرف برقم الهاتف أو username. انظر [Telegram ID مقابل username]({{< relref "telegram-id-vs-username.md" "en" >}}).

### لماذا لا يرد البوت في مجموعتي؟
يرد البوت فقط عند الرد عليه أو إرسال /id أو /help أو ذكر @print_id_bot. انظر [البوت لا يرد في المجموعة]({{< relref "bot-not-responding-in-group.md" "en" >}}) لقائمة كاملة.

### هل البوت مجاني؟
نعم. لا يتطلب تسجيلاً أو دفعة.

### ما الفرق بين user ID وchat_id؟
**User ID** يحدد حساب تلغرام. **chat_id** يحدد محادثة (دردشة خاصة، مجموعة، مجموعة فائقة، قناة). انظر [Telegram ID مقابل username]({{< relref "telegram-id-vs-username.md" "en" >}}).
