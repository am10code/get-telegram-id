تلغرام گروپ یا سپرگروپ کا **chat_id** Bot API انٹیگریشنز کے لیے درکار ہے۔ تیزی سے حاصل کرنے کا طریقہ یہاں ہے۔

## قدم 1: بٹ کو اپنے گروپ میں شامل کریں

1. تلغرام میں [@print_id_bot](https://t.me/print_id_bot) کھولیں۔
2. **شروع** پر ٹیپ کریں۔
3. بٹ کو گروپ میں شامل کریں: گروپ کھولیں → ممبران شامل کریں → `print_id_bot` تلاش کریں → شامل کریں۔

جب بٹ شامل ہو جاتا ہے تو گروپ نام اور **chat_id** کے ساتھ خوش آمدید پیغام بھیجتا ہے۔ وہاں سے کاپی کر سکتے ہیں۔

## قدم 2: اگر دوبارہ حاصل کرنا ہو

بٹ گروپوں میں صرف تب جواب دیتا ہے جب:

- آپ بٹ کے پیغام کا جواب دیتے ہیں؛
- آپ `/id` یا `/help` بھیجتے ہیں (@print_id_bot کے ساتھ یا بغیر)؛
- آپ @print_id_bot کے ساتھ کوئی کمانڈ بھیجتے ہیں (مثلاً `/foo@print_id_bot`)؛
- آپ پیغام میں @print_id_bot کا ذکر کرتے ہیں۔

`/id` بھیجیں یا @print_id_bot کا ذکر کریں؛ بٹ گروپ نام اور chat_id کے ساتھ جواب دے گا۔ کاپی کرنے کے لیے نمبر پر ٹیپ کریں۔

## گروپ بمقابلہ سپرگروپ: chat_id

گروپ اور سپرگروپ ID اکثر **منفی نمبر** ہوتے ہیں۔ یہ عام ہے۔ جب گروپ سپرگروپ میں اپ گریڈ ہوتا ہے تو chat_id بدل سکتا ہے۔ تفصیل: [گروپ بمقابلہ سپرگروپ chat_id]({{< relref "telegram-group-chat-id-vs-supergroup.md" "en" >}}).

## خرابیوں کا ازالہ

اگر بٹ جواب نہیں دیتا تو چیک کریں:

- بٹ کو پیغامات پڑھنے کی اجازت ہے؛
- آپ نے اسے درست ٹرگر کیا (جواب، /id، /help یا @print_id_bot)؛
- بٹ پرائیویسی موڈ میں نہیں جو گروپ پیغامات بلاک کرتا ہے (اگر لاگو ہو)۔

## FAQ

### میری گروپ کا chat_id منفی کیوں ہے؟
گروپ اور سپرگروپ ID Telegram Bot API میں عام طور پر منفی ہوتے ہیں۔ یہ ڈیزائن کے مطابق ہے۔ دیکھیں [تلغرام chat_id فارمیٹ]({{< relref "telegram-chat-id-format.md" "en" >}}).

### کیا میں کسی اور کی گروپ کا chat_id حاصل کر سکتا ہوں؟
صرف ان گروپوں کا جن میں آپ ممبر ہیں۔ بٹ کو گروپ میں شامل کریں اور /id بھیجیں یا اس کا ذکر کریں۔

### کیا بٹ سپرگروپوں میں کام کرتا ہے؟
جی ہاں۔ گروپ اور سپرگروپ دونوں سپورٹ ہیں۔ بٹ گروپ نام اور chat_id دکھاتا ہے۔ دیکھیں [گروپ بمقابلہ سپرگروپ chat_id]({{< relref "telegram-group-chat-id-vs-supergroup.md" "en" >}}).

### اگر بٹ گروپ سے ہٹا دیا گیا تو کیا کریں؟
دوبارہ شامل کریں۔ شامل کرنے پر وہ chat_id کے ساتھ خوش آمدید پیغام بھیجتا ہے۔ /id بھیج کر یا @print_id_bot کا ذکر کر کے بھی حاصل کر سکتے ہیں۔

### کیا گروپ اور سپرگروپ کے لیے chat_id ایک جیسا ہے؟
سپرگروپ میں مائگریشن کے بعد chat_id بدل سکتا ہے۔ ہمیشہ بٹ یا API سے موجودہ ویلیو استعمال کریں۔
