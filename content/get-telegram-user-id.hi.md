आपका **Telegram user ID** आपके खाते का संख्यात्मक पहचानकर्ता है। यह Telegram ऐप में दिखाई नहीं देता, लेकिन बॉट और इंटीग्रेशन इसे पढ़ सकते हैं जब आप उनके साथ इंटरैक्ट करते हैं। इसे सुरक्षित रूप से प्राप्त करने के तरीके यहाँ हैं।

## विधि 1: @print_id_bot का उपयोग करें (सबसे तेज़)

1. Telegram में [@print_id_bot](https://t.me/print_id_bot) खोलें।
2. **स्टार्ट** पर टैप करें।
3. बॉट आपके **User ID** और **Chat ID** के साथ जवाब देता है (प्राइवेट चैट में वे मेल खाते हैं)।
4. कॉपी करने के लिए नंबर पर टैप करें।

बॉट आपकी Telegram भाषा सेटिंग के अनुसार 16 भाषाओं में जवाब देता है। असमर्थित भाषाएँ अंग्रेज़ी पर वापस जाती हैं।

## विधि 2: बॉट को कोई भी संदेश भेजें

@print_id_bot के साथ प्राइवेट चैट में कोई भी संदेश, फोटो, स्टिकर या दस्तावेज़ भेजें। बॉट आपके User ID और Chat ID के साथ जवाब देता है। अल्प जवाब के लिए `/id` भी इस्तेमाल कर सकते हैं।

## विधि 3: संदेश फॉरवर्ड करें (और सीमाएँ)

यदि आप किसी अन्य उपयोगकर्ता का संदेश @print_id_bot को फॉरवर्ड करते हैं, तो बॉट मूल भेजने वाले का User ID दिखा सकता है। हालाँकि, यदि भेजने वाले की प्राइवेसी सेटिंग फॉरवर्ड किए जाने पर उनका ID छिपाती है, तो आपको "ID hidden" दिखेगा। फोन नंबर या username से किसी और का ID विश्वसनीय रूप से नहीं मिल सकता। विवरण: [फॉरवर्ड संदेश और छिपे ID]({{< relref "telegram-forwarded-message-id.md" "en" >}}).

## आप क्या नहीं कर सकते

- **फोन नंबर से खोज**: केवल फोन नंबर से Telegram user ID नहीं मिल सकता। ऐसा दावा करने वाली सेवाएँ अक्सर घोटाला या प्राइवेसी उल्लंघन होती हैं।
- **username से खोज**: Username ID नहीं हैं। ID पाने के लिए उपयोगकर्ता को बॉट या ऐप के साथ इंटरैक्ट करना होगा।
- **इंटरैक्ट के बिना दूसरों के ID देखना**: आप केवल उन चैट के ID प्राप्त कर सकते हैं जिनमें आप हैं या जो उपयोगकर्ता आपको मिलने वाले संदेश फॉरवर्ड करते हैं (और फॉरवर्ड की अनुमति देते हैं)।

## सामान्य प्रश्न

### क्या मेरा Telegram user ID निजी है?
यह Telegram UI में दिखाई नहीं देता, लेकिन बॉट और ऐप इंटरैक्ट करते समय इसे पढ़ सकते हैं। अविश्वसनीय सेवाओं के साथ साझा न करें।

### क्या मैं अपना Telegram user ID बदल सकता हूँ?
नहीं। यह एक बार नियत होता है और नहीं बदलता।

### फॉरवर्ड संदेशों के लिए बॉट "ID hidden" क्यों दिखाता है?
भेजने वाले ने प्राइवेसी सेटिंग चालू की हैं जो फॉरवर्ड होने पर उनका ID छिपाती हैं। देखें [फॉरवर्ड भेजने वाले का ID छिपा]({{< relref "telegram-privacy-forwarded-id-hidden.md" "en" >}}).

### क्या @print_id_bot का उपयोग सुरक्षित है?
हाँ। बॉट केवल उन ID दिखाता है जिन तक आपकी पहुंच है (आपका स्वयं या अपने चैट्स से)। यह Telegram द्वारा दिए गए से अधिक आपका डेटा स्टोर या शेयर नहीं करता।

### Telegram user ID कैसा दिखता है?
संख्यात्मक मान, अक्सर 9–10 अंक। उदाहरण: `123456789`.
